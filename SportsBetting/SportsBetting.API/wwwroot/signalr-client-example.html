Usage Instructions

  1. Make sure your API is running at http://localhost:5192
  2. Click Connect to establish WebSocket connection
  3. Enter an Outcome ID (GUID) and click Subscribe to Outcome
  4. Open another browser tab/window and place bets via API to see live updates
  5. Watch the order book update in real-time as bets are placed, matched, or cancelled

Example curl command to place a bet:

curl -X POST http://localhost:5192/api/bets/exchange \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -d '{
    "outcomeId": "YOUR_OUTCOME_ID",
    "stake": 100,
    "odds": 2.5,
    "side": "Back"
  }'
